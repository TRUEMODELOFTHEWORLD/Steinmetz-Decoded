## Citation And Data Export

The archive now publishes reusable data files for researchers, contributors, and external tools. These exports are designed for traceability: they expose source IDs, review states, generated indexes, and citation records without pretending candidate data is already canonical.

<div class="source-text-warning" data-layer="source">
  <strong>Use rule:</strong> cite generated data as an archive aid. Cite exact Steinmetz claims only after checking the source page, scan, and review status.
</div>

## Review Boundary

Data exports preserve current review status. They should help researchers find and reuse the archive's work, but they do not remove the need for source verification, especially for equations, direct quotations, figure captions, patent claims, and interpretive comparisons.
