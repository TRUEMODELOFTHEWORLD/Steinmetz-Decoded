This is not a marketing plan. It is a public-service discoverability plan: help people searching for Charles Proteus Steinmetz, alternating-current theory, hysteresis, transients, symbolic method, impedance, reactance, electric waves, and historical electrical terminology find source-grounded material.

## What The Site Provides

- A canonical public home: [https://steinmetz-decoded.vercel.app/](https://steinmetz-decoded.vercel.app/).
- A generated sitemap at `/sitemap-index.xml`.
- A crawler-friendly `robots.txt` pointing to the sitemap.
- Open Graph and social preview metadata.
- JSON-LD structured data describing the website, Steinmetz, the publisher, the public corpus dataset, and key research entry points.
- A crawler-friendly [Steinmetz Research Topic Map](/steinmetz-research-topics/) that links major people, books, concepts, equations, diagrams, and source readers.
- `llms.txt`, `humans.txt`, and a public data manifest for AI agents, researchers, and maintainers.
- Public data manifests, source-text readers, diagrams, equations, and concept dossiers.
- Specific generated descriptions for source-text pages instead of generic generated-page snippets.

## Outside-The-Site Actions

1. Add the site to Google Search Console and submit `https://steinmetz-decoded.vercel.app/sitemap-index.xml`.
2. Add the same sitemap to Bing Webmaster Tools.
3. Put the Vercel URL in the GitHub repository website field and repository description.
4. Add useful GitHub repository topics such as `steinmetz`, `electrical-engineering`, `history-of-science`, `alternating-current`, `hysteresis`, `transients`, `public-domain`, and `research-archive`.
5. Share specific pages when useful: concept pages, source-text readers, equations, diagrams, and visual galleries.
6. Link to the topic map when introducing the project because it gives readers and crawlers one complete public index.
7. Invite corrections through GitHub issues so trust compounds over time.
8. If a custom domain is added later, keep one canonical host and redirect the other domain.

## Honest Search Phrases

The archive can serve searches such as:

- Charles Proteus Steinmetz writings
- Steinmetz alternating current theory
- Steinmetz hysteresis law
- Steinmetz transient electric phenomena
- Steinmetz symbolic method complex quantities
- Steinmetz impedance reactance admittance
- Steinmetz electric waves radiation light illumination
- Steinmetz ether field language
- Steinmetz equations and diagrams
- Steinmetz public domain books
- historical electrical engineering terminology

## Boundary

Do not chase traffic by weakening the archive. The strongest discoverability signal is durable usefulness: accurate titles, stable URLs, source text, diagrams, equations, citations, and clear labels separating source fact from interpretation.
