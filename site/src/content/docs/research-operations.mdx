Most readers should not start here. This page gathers the machinery that keeps the archive honest: source custody, workbenches, ledgers, verification queues, publication checks, data exports, and completion tracking.

If you came to read Steinmetz, start with [Start Reading](/start-reading/) or [Browse Books](/book-coverage/). If you came to verify, improve, cite, or extend the archive, this is the right room.

## Principle

The operations layer exists so the reading layer can be clean. Readers should see books, concepts, diagrams, and math first. Contributors should still be able to audit every claim, passage, equation, and figure when needed.
