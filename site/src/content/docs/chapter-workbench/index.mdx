## How To Use The Workbench

1. Pick a source and section.
2. Read the source excerpt and theme snippets.
3. Open the full source text and scan/PDF links.
4. Promote verified passages into concept, equation, diagram, glossary, hidden-gem, and comparison pages.
5. Keep source wording, modern engineering explanation, and ether-field interpretation separated by layer.
