# Public Discoverability Guide

This project is not trying to sell anything. Discoverability here means helping students, engineers, historians, and independent researchers find a source-grounded Steinmetz archive when they search for Steinmetz or the electrical concepts he helped formalize.

## What The Site Now Provides

- A canonical public URL: https://steinmetz-decoded.vercel.app/
- A generated sitemap at `/sitemap-index.xml`.
- A crawler-friendly `robots.txt` pointing to the sitemap.
- Open Graph and Twitter preview metadata.
- JSON-LD structured data describing the website, Steinmetz as the subject, and the archive as a free research collection.
- `llms.txt` for AI/research agents that need a compact map of the archive.
- `humans.txt` explaining the public-service purpose of the project.
- A public data manifest and source-grounded pages that preserve verification boundaries.

## What To Do Outside The Codebase

1. Add the site to Google Search Console and submit `https://steinmetz-decoded.vercel.app/sitemap-index.xml`.
2. Add the site to Bing Webmaster Tools and submit the same sitemap.
3. Link to the archive from the GitHub repository description and README website field.
4. When sharing online, link to specific pages, not only the homepage: concept pages, source-text readers, equation pages, and visual galleries.
5. Ask relevant public-domain/history/electrical-engineering communities to cite or link to the pages only where genuinely useful.
6. Encourage corrections through GitHub issues so the site earns trust over time.
7. If a custom domain is added later, redirect the Vercel domain to it and keep one canonical host.

- Charles Proteus Steinmetz writings
- Steinmetz alternating current theory
- Steinmetz hysteresis law
- Steinmetz transient electric phenomena
- Steinmetz symbolic method complex quantities
- Steinmetz impedance reactance admittance
- Steinmetz electric waves radiation light illumination
- Steinmetz ether field language
- Steinmetz electrical engineering diagrams
- historical electrical engineering terminology

## Editorial Rule

Do not chase traffic by weakening the archive. The best long-term discoverability signal is honest, source-grounded usefulness: clear titles, accurate descriptions, stable URLs, citations, diagrams, equations, and explicit labels for source fact versus interpretation.
